# PastillaMe

Medication reminder app built with Flutter.

## Brand

- **Name:** PastillaMe
- **Color:** Orange (#FF6D00)

## Stack

- **Flutter** (>=3.0.0) — Cross-platform mobile app
- **Google Maps** — Location & mapping features
- **Camera** — Photo capture & processing
- **HTTP** — API communication

## Quick Start

```bash
flutter pub get
flutter run
```

Requires Google Maps API key.
